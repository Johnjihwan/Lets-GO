# Let's GO!
## 요즘 대세인 언어 알고는 가자!

### What is Go?
> Go는 간결하고 신뢰성이 있으며 효율적인 소프트웨어를 손쉽게 만들기 위한 오픈소스 프로그래밍 언어이다.

## Go의 특징
> Go는 빠른 성능과 안정성, 편의성을 제공하기 때문에 쉽게 프로그래밍이 가능하다.

### 간결한 문법
> 문법들이 간결하고 키워드의 수가 적다.  
> golang은 상대적으로 키워드 수가 적기 때문에 상당히 간결하고 배우는데 시간이 오랜시간이 걸리지 않는다.  

### 개발 속도 및 컴파일 언어
> 인터프리터 방식의 언어가 아닌 컴파일 방시의 언어로 개발하게 되면 애플리케이션을 컴파일하면 정말 많은 시간이 걸리게 된다.  
> golang 은 간결한 의존석 해석 알고리즘을 통해서 다른 컴파일 언어에 비해 빠르게 동작하는 컴파일러를 제공한다.  
> golang에서는 참조를 하는 라이브러리만을 해석한다는 것.

### 동적 + 정적 언어
> golang 은 컴파일 기반 정적 타입 언어 입니다. 하지만 동적 언어의 특성도 수용하기 때문에 golang은 컴파일러의 안정성을 보장 받으면서 동적언어의 유연함과 자유도를 얻을 수 있다는 것 입니다.

### 가비지 컬렉션(GC - 메모리관리)
> golang에는 메모리를 직접관리할 필요가 없습니다 왜냐하면 가비지 컬렉션이 존재하기 때문입니다.  
> C, C++에서는 메모리를 할당하면 메모리를 해제 해주어야 합니다. 로직 작성보다 관리에 더 많은 노력과 시간을 소모하게되어 생산성이 떨어지죠.  
> 그래서 메모리를 알아서 관리해주는 가비지 컬렉션 기술이 나왔고 java c#에 등장하게 된것이다.
